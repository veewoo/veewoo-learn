# Veewoo Learn

A Next.js application for learning and skill development with an intuitive interface and comprehensive learning paths.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Generate TypeScript types for Supabase:
```bash
npx dotenv -e .env.local -- npm run supabase:generate
```

Add Shadcn components:
```bash
npx shadcn@latest add button
```
